About Splitscreen

Splitscreen is a unique, mouse-responsive fullscreen billboard-style jQuery overlay plugin with three modes: splash, promo, and teaser; and several options for plugin customization. Simply add the Splitscreen Css and Javascript files to the head of your webpage and initialize the plugin at the bottom of your webpage, leave the rest of your webpage content in tact and Splitscreen will do the rest.

In Splash mode, Splitscreen appears above your website content and is dismissed(closed) by clicking on the center of the page.

In Promo mode Splitscreen appears above your website content with two buttons behind the split. One button leades to a custom url, while the other dismisses(closes) Splitscreen, allowing the user to view the current page.

In Teaser mode Splitscreen appears above your website content and provides the option to add custom 'teaser' html behind the split; in this mode Splitscreen cannot be dismissed(closed) unless you add css to the 'teaser' html or use javascript to Close Splitscreen Externally.

1. Splitscreen Assets - top

The Splitscreen directory contains three folders and comes with three example files to show you how jQuery.Splitscreen is used. The plugins' assets are as follows:

/js

The js directory includes the following javascript files.

» jquery.splitscreen.1.3.js

	This is the main javascript file for the Splitscreen plugin.
	The minified version is also included in the package: jquery.splitscreen.1.3.min.js
» jquery-1.6.2.min.js
	The latest version of the jQuery library.
» jquery.easing.1.3.js
	The jQuery plugin for animation easing.
» mdetect.js
	Detects mobile devices. Referenced in jquery.splitscreen.1.3.js to prevent
	the plugin from displaying on incompatiable mobile devices so that the
	webpage can be viewed unobstructed in those situations.

/css

The css directory includes the stylesheet for the plugin in one file:

» splitscreen.css

	The css file for the Splitscreen plugin.

/img

The img directory contains the numerous images that are native to the plugin.

2. Files to Include in the Header - top

Include the follow css and javascript files in the head of your html document.


	<!-- CSS -->
	<link rel="stylesheet" href="css/splitscreen.css" type="text/css" media="all"/>
	
	<!-- JS -->
	<script type="text/javascript" src="js/jquery-1.6.2.min.js"></script>
	<script type="text/javascript" src="js/jquery.easing-1.3.js"></script>
	<script type="text/javascript" src="js/mdetect.js"></script>	
	<script type="text/javascript" src="js/jquery.splitscreen-1.3.js"></script>

3. Initializing the Plugin - top

DEFAULT

To initialize the plugin with the default options, after all of your content, add the following code to the end of your html document, right above the closing </body> tag:

		
	<script type="text/javascript">
	
	$('body').splitscreen();
	
	</script>

CUSTOM (recommended)

The example html files use the custom approach. To initialize the plugin with the custom options, add the following code to the end of your html document after all of your content, right above the closing </body> tag and modify according to your requirements.See Configuration Options for parameter info.

4) Close Splitscreen Externally - top

You can close Splitscreen externally via javascript function or by adding a special css class to an element.

A. Via javascript call, invoke this function in your javascript code: $.closeSplitscreen();

B. Via css element action add the class x-splitscreen to an element. For Example: 

	<a href='#' class="x-splitscreen">Continue to website</a>
